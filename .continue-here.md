# BLOCKING CONSTRAINTS — Read Before Anything Else

> These are not suggestions. Each constraint below was discovered through failure.
> Acknowledge each one explicitly before proceeding.

- [ ] CONSTRAINT: Playwright bundled Chromium requires glibc — Alpine uses musl. MUST use system Chromium via `PLAYWRIGHT_CHROMIUM_EXECUTABLE_PATH` env var.
- [ ] CONSTRAINT: `page.content()` captures HTML BEFORE JS renders pricing tables. MUST have `waitForLoadState('networkidle')` + timeout.
- [ ] CONSTRAINT: MIMO API can return HTTP 200 with empty/null content. MUST validate `result.text` is non-empty before parsing.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Wrong pricing URLs | claude.com/pricing is subscription page, not API pricing. platform.deepseek.com returns 403. fireworks.ai/pricing only has embeddings. | blocking | Always verify URL contains per-token API pricing, not subscription/embedding/GPU pricing |
| Empty AI response → silent success | generateText() returns empty string without throwing. Parser returns empty content. DB accepts empty string (NOT NULL allows ''). | blocking | Validate response.text is non-empty after each generateText() call |
| 0-model extraction → silent success | Adapter returns models:[] and job completes with no warning. Impossible to debug. | blocking | Log warning when normalized.models.length === 0 |
