# Phase 11 Plan: Digest & Free Offers Enhancement

## Goal

Redesign the digest page to prominently display free models and promotions with direct provider links, replacing the current text-based approach with structured card components.

## Success Criteria

1. Free models displayed as prominent cards with green "FREE" badge
2. Each card links directly to provider pricing page
3. Promotions section shows discounts with provider links
4. No raw markdown formatting visible in UI
5. Mobile responsive design
6. Works in both light and dark themes

## Technical Approach

### Data Source
Instead of parsing markdown at runtime, query the `promotions` table directly:
- Join with `sources` table for provider URLs
- Filter by date (today's promotions)
- Include free tiers (`type = 'free_tier'`) and promotions (`type = 'promotion'`)

### Component Architecture

```
DigestArticle.tsx
├── FreeOffersSection (new)
│   └── FreeOfferCard (new) — for each free model
├── PromotionsSection (new)
│   └── PromotionCard (new) — for each discount
└── ReactMarkdown (existing)
```
