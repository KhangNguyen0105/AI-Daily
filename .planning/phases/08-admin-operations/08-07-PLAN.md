     - GET handler:
       - Accept optional query params: status (active/inactive/all), type (HTTP/Browser/all)
       - Build conditions array:
         - If status === 'active': eq(sources.isActive, 1)
         - If status === 'inactive': eq(sources.isActive, 0)

     - Import auth, z, NextResponse
     - Auth check at top
     - PATCH handler:
       - Extract id from params
       - Validate: z.object({ isActive: z.boolean() })
       - Update: db.update(sources).set({ isActive: isActive ? 1 : 0, updatedAt: new Date() }).where(eq(sources.id, id))
       - Fetch updated source name for toast message
       - Return { success: true, name: sourceName, isActive }
