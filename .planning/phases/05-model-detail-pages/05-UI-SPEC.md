# Phase 5 — Model Detail Pages: UI Design Contract

> Visual and interaction contract for the model detail page. Generated by gsd-ui-researcher based on existing codebase patterns and upstream decisions.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Tailwind CSS 4.x only) |
| Preset | not applicable |
| Component library | none (custom components) |
| Icon library | none (text labels + Unicode arrows) |
| Font | system default (`font-sans`) |

**Rationale:** The project uses Tailwind CSS 4.x with `@import "tailwindcss"` and no explicit theme tokens. No shadcn/ui, no component library. All components are hand-built with Tailwind utility classes. This phase continues that approach.

## Interaction Patterns

### Currency Toggle

**Location:** Above the PricingGrid, aligned left.

**Behavior:**
- Two-button group: USD | VND
- Active: `bg-blue-600 text-white border-blue-600`
- Inactive: `bg-white text-gray-700 border-gray-300 hover:bg-gray-50`
- State managed by `ModelDetailClient` via `useState<'usd' | 'vnd'>('usd')`
- Passed to `PricingGrid` as `currency` prop
- Uses same pattern as `HomePageClient` and `PricingTable`

**ARIA:** `role="group"`, `aria-label="Toggle currency"`

### Link Behavior

| Link | Destination | Behavior |
|------|-------------|----------|
| "Back to pricing" | `/` | Client-side navigation via `next/link` |
| Model name (in PricingTable) | `/model/[slug]` | Client-side navigation via `next/link` |
| Provider docs/API/playground | External URLs | `target="_blank" rel="noopener noreferrer"` |
| Promotion source URL | External URL | `target="_blank" rel="noopener noreferrer"` |

### Hover States

| Element | Hover Style |
|---------|-------------|
| Links | `hover:text-blue-800 hover:underline` |
| Currency toggle (inactive) | `hover:bg-gray-50` |
| Table rows (PricingTable) | `hover:bg-gray-50 transition-colors` |

### Navigation Flow

```
Landing Page (/)
  └── User clicks model name in PricingTable
        └── /model/[slug] (Model Detail Page)
              └── User clicks "Back to pricing"
                    └── / (Landing Page)
```

## Data Flow Diagram

```
Server Component (app/model/[slug]/page.tsx)
  │
  ├─ generateStaticParams() → DB query for all model slugs
  ├─ resolveSlug(slug) → { modelName, sourceId }
  ├─ DB: latest extraction (JOIN sources)
  ├─ DB: price history (extractions WHERE model + source)
  ├─ DB: promotions (WHERE sourceId, try/catch for missing table)
  ├─ getLatestExchangeRate()
  │
  └─→ <ModelDetailClient model={...} history={...} promotions={...} exchangeRate={...} />
        │
        ├─ useState<'usd' | 'vnd'>('usd')  ← currency state
        │
        ├─ Hero Section (model name, provider badge, confidence, back link)
        ├─ PricingGrid (3 cards, currency-aware)
        ├─ PriceHistoryChart (Recharts dual-line)
        ├─ Specifications (context window, family, release date)
        ├─ PromotionsList (active/expired cards)
        ├─ ProviderLinks (docs, API, playground)
        └─ Digest Mentions (placeholder)
```
