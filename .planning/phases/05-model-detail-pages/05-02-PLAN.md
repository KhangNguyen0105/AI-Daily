  <action>
    Create the directory `app/model/[slug]/` and the file `page.tsx`.

    This is a server component (NO 'use client' directive). It follows the same pattern as app/page.tsx:
    1. Server fetches data via Drizzle
    2. Wraps in try/catch for build safety
    3. Passes data to a client component

    Per D-01: Use `generateStaticParams` for known models with ISR revalidation.

    Structure:

    ```
    export const revalidate = 60; // ISR: revalidate every 60 seconds
    ```

    generateStaticParams():
    - Query: SELECT DISTINCT modelName, sourceId FROM extractions GROUP BY modelName, sourceId
    - Map each row to { slug: generateSlug(row.modelName, row.sourceId) }
    - Wrap in try/catch, return [] if DB unavailable during build
    - Import generateSlug from '@/app/lib/slug'

    Default export async function ModelDetailPage({ params }):
    - params type is `Promise<{ slug: string }>` (Next.js 16 async params — VERIFIED in research)
    - MUST await params: `const { slug } = await params;`
    - Call resolveSlug(slug) to get { modelName, sourceId }
    - If null, call notFound() from 'next/navigation'
    - Fetch latest extraction (JOIN with sources):
      ```
      db.select({
        id: extractions.id,
        modelName: extractions.modelName,
        inputPricePer1m: extractions.inputPricePer1m,
        outputPricePer1m: extractions.outputPricePer1m,
        contextWindow: extractions.contextWindow,
        confidence: extractions.confidence,
        collectedAt: extractions.collectedAt,
        sourceId: extractions.sourceId,
        sourceName: sources.name,
        sourceUrl: sources.url,
      })
      .from(extractions)
      .leftJoin(sources, eq(extractions.sourceId, sources.id))
      .where(and(eq(extractions.modelName, modelName), eq(extractions.sourceId, sourceId)))
      .orderBy(desc(extractions.collectedAt))
      .limit(1)
      ```
    - If no result, call notFound()
    - Fetch price history (D-05, D-06, D-08: all history for this model + source):
      ```
      db.select({
        collectedAt: extractions.collectedAt,
        inputPricePer1m: extractions.inputPricePer1m,
        outputPricePer1m: extractions.outputPricePer1m,
      })
      .from(extractions)
      .where(and(eq(extractions.modelName, modelName), eq(extractions.sourceId, sourceId)))
      .orderBy(extractions.collectedAt)
      ```
    - Fetch promotions (D-09, wrap in try/catch — table may not exist):
      ```
      db.select()
        .from(promotions)
        .where(eq(promotions.sourceId, sourceId))
      ```
    - Fetch exchange rate (same pattern as app/page.tsx):
      ```
      let exchangeRate = FALLBACK_RATE;
      try { exchangeRate = await getLatestExchangeRate(); } catch { /* fallback */ }
      ```
    - Return <ModelDetailClient model={latest} history={history} promotions={activePromotions} exchangeRate={exchangeRate} />

    Imports needed:

    - notFound from 'next/navigation'
    - getLatestExchangeRate, FALLBACK_RATE from '@/src/pipeline/exchange-rate-worker'
