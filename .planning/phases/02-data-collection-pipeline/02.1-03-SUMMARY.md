## Decisions Made

| Decision | Rationale |
|----------|-----------|
| 6 independent confidence dimensions (D-07) | Enables tooltip to explain "Source high, Extraction medium, Freshness low = Medium overall" |
| Confidence weights: 0.25/0.25/0.20/0.20/0.10 (D-07) | Source and extraction most important; freshness least (data can be slightly stale) |
| Critical rules cap overall score (D-07) | Low price confidence caps at Medium(54); unofficial caps at High(74); disagreement->quarantined |
| Per-field confidence (D-07) | Row can have reliable model name (high) but uncertain context window (low) |
| Human override preserves machine scores (D-07) | Audit trail: machine score kept for comparison, override replaces overall in UI |
| Tiered SLA enforcement (D-03) | Tier1 providers change pricing frequently (24h), others less so (48h/7d) |
| Freshness badges: green/blue/amber/red (D-03) | Simple visual indicator for data freshness |
| >30d data excluded from rankings (D-03) | Prevents stale data from appearing in comparisons |
