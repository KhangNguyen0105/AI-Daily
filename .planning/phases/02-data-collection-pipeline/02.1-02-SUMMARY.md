## Decisions Made

| Decision | Rationale |
|----------|-----------|
| Evidence-first verification (D-08) | No price verified without source anchor — prevents hallucinated prices |
| SHA256 hashing of evidence quotes | Immutable audit trail; changes detected via hash mismatch |
| 14 edge case types | Covers all non-standard pricing patterns per D-08 specification |
| isComparableTokenPricing filter | Separates token-based from non-comparable pricing for rankings |
| Numeric tolerance: 0.1% same, 0.5% converted, 0.05% large change | Balances precision with real-world formatting variations |
| Human review fields on extractions | Admin override capability for automated verification decisions |
