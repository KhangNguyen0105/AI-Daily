       - Return updated article
     - Export `GET` and `PUT` (Next.js App Router route handler convention)

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-07 | Tampering | Article edit API | mitigate | Zod validation on all inputs. Middleware ensures only authenticated admin can access. |
| T-08-08 | Information Disclosure | Article versions | accept | Article content is already published publicly. Version history is admin-only. |
| T-08-09 | Denial of Service | Version creation | accept | Single admin user, low volume. No rate limiting needed. |
| T-08-10 | Tampering | Rollback action | mitigate | Confirmation dialog required. Rollback itself creates a version (audit trail). |
</threat_model>
