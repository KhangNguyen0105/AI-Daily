## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-09 | Tampering | Article edit API | mitigate | Zod validation on all inputs. Middleware ensures only authenticated admin can access. |
| T-08-10 | Information Disclosure | Article versions | accept | Article content is already published publicly. Version history is admin-only. |
| T-08-11 | Denial of Service | Version creation | accept | Single admin user, low volume. No rate limiting needed. |
| T-08-12 | Tampering | Rollback action | mitigate | Confirmation dialog required. Rollback itself creates a version (audit trail). Transaction ensures atomicity. |
</threat_model>
