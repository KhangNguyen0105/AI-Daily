## Decisions Made

| Decision | Rationale |
|----------|-----------|
| 4-hour cron for Tier 1 refresh (D-03) | Simplest cron pattern within 2-4 hour target; 6 runs/day provides good freshness |
| Priority=1 for Tier 1, priority=10 for routine (D-01) | BullMQ lower number = higher priority; ensures Tier 1 is processed first |
| CNY stored as-is, conversion at score stage | Allows exchange rate to be applied at processing time, not extraction time |
| Added normalize() to all Tier 1 adapters | Preserves raw_price_text alongside normalized values per D-05 |
| Fixed PromotionResult credits type | Pre-existing Zod schema allows null but interface didn't; Rule 1 fix |
