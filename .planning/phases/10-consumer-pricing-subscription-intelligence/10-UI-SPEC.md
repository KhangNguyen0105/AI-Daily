### Sort Controls

| Sort Option | Value | Behavior |

│ [Provider Logo]  Plan Name          │  ← provider_name + plan_name

**Card styling:**

**Source link:**

## Interaction Patterns

### Filter Pills

- Click toggles active filter
- Only one filter active at a time
- Resets sort to default when filter changes
- Active state: `bg-accent-blue text-white`
- Inactive state: `bg-bg-tertiary text-text-primary hover:bg-bg-secondary`

### Sort

- Dropdown select or button group
- Default: Price Low to High
- Persists during filter changes (unless filter makes sort irrelevant)

### Card Hover

- `hover:shadow-md transition-shadow` on card container
- Source link: `hover:text-accent-blue/80`

### Navigation

- "Subscriptions" link added to TopNav between "Promotions" and "Compare"
- Active detection: `pathname.startsWith('/subscriptions')`

## Data Shape

```typescript
interface SubscriptionPlan {
  id: number;
  providerName: string;       // e.g. "OpenAI"
  planName: string;           // e.g. "ChatGPT Plus"
  monthlyPrice: number | null; // USD
  annualPrice: number | null;  // USD total annual
  annualMonthlyPrice: number | null; // effective monthly when billed annually
  freeTrialDays: number | null;
  freeTrialConditions: string | null;
  keyFeatures: string[];       // JSONB array
  currency: string;            // "USD"
  sourceUrl: string | null;
  startDate: Date | null;
  endDate: Date | null;
}
```

## Accessibility

- All interactive elements must have visible focus indicators
- Filter pills: `focus:outline-none focus:ring-2 focus:ring-accent-blue focus:ring-offset-2`
- Source links: standard anchor accessibility (`target="_blank"` with `rel="noopener noreferrer"`)
- Cards: use semantic HTML (`<article>` or `<div>` with `role="article"`)
- Free trial badge: `aria-label="Free trial available for {N} days"` for screen readers
