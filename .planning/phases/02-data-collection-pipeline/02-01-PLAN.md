**Each adapter.ts pattern:**
- Import ProviderAdapter, CrawlResult, ExtractionResult from '../base'
- Import config from './config'
- Define a provider-specific Zod schema: `z.object({ models: z.array(z.object({ modelName: z.string(), inputPricePer1m: z.number(), outputPricePer1m: z.number(), contextWindow: z.number() })) })`
- Class extends ProviderAdapter with `config = providerConfig`
- Override crawl() with PlaywrightCrawler (same as OpenAI pattern)
- extract(html): truncate to 100K chars, call generateObject with provider-specific prompt, map to ExtractionResult[] with confidence 'likely'
- normalize(extractions): return extractions.map(e => ({ ...e, confidence: 'likely' as const }))

**Provider-specific extraction prompt guidance:**
- Anthropic: "Extract Claude model pricing. Look for model names like claude-3.5-sonnet, claude-3-opus, etc. Input/output prices per million tokens."
- Google: "Extract Gemini model pricing. Look for gemini-2.0-flash, gemini-1.5-pro, etc. Note tiered pricing if present."
- Mistral: "Extract Mistral model pricing. Look for mistral-large, mistral-small, codestral, etc."
- Cohere: "Extract Cohere model pricing. Look for command-r-plus, command-r, command-light, etc. Convert any non-per-token pricing to per-1M-tokens."
- Groq: "Extract Groq model pricing. Look for llama, mixtral, gemma models with Groq pricing."
- Together: "Extract Together AI model pricing. Look for various open-source models with Together pricing."
- Perplexity: "Extract Perplexity Sonar model pricing. Look for sonar, sonar-pro, sonar-reasoning, etc."
- xAI: "Extract xAI Grok model pricing. Look for grok-2, grok-2-mini, grok-3, etc."
- Fireworks: "Extract Fireworks AI model pricing. Look for various models with Fireworks pricing."
- DeepSeek: "Extract DeepSeek model pricing. Look for deepseek-chat, deepseek-coder, deepseek-reasoner, etc."
- Bedrock: "Extract Amazon Bedrock model pricing. Look for models from multiple providers available through Bedrock. Prices may be per-1000-tokens; convert to per-1M-tokens."

**normalize() for DeepSeek and Bedrock:**
- DeepSeek prices may be in per-1K-tokens format; normalize to per-1M-tokens (multiply by 1000)
- Bedrock prices may be in per-1K-tokens or per-1000-tokens format; normalize to per-1M-tokens
