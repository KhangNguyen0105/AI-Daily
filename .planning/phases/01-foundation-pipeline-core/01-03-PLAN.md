## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Worker → External site | Crawlee fetches from openai.com -- untrusted HTML stored in raw_data.evidence |
| Worker → AI API | HTML sent to LLM for extraction -- response validated by Zod schema |
| DB → Landing page | Extractions displayed to public -- no sensitive data exposed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | AI extraction output | mitigate | Zod schema in OpenAI adapter validates extraction shape before DB insert |
| T-03-02 | Information Disclosure | raw_data.evidence | accept | HTML is public pricing page content; no PII risk |
| T-03-03 | Denial of Service | BullMQ workers | mitigate | maxRequestsPerCrawl: 1, concurrency: 1 per worker, exponential backoff |
| T-03-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages; all verified in Plan 01 |
</threat_model>
