  <action>
Create `src/pipeline/article-diff.ts` with the following:

Export a `DiffResult` interface:
- `newModels`: array of `{ modelName: string; inputPricePer1m: number | null; outputPricePer1m: number | null; contextWindow: number | null; sourceName: string }`
- `priceChanges`: array of `{ modelName: string; field: 'input' | 'output'; oldPrice: number; newPrice: number; changePercent: number }`
- `newPromotions`: array of `{ modelPattern: string; description: string; type: string }`
- `totalModelsToday`: number

Export an async `computeDiff(today: Date, yesterday: Date)` function that:
1. Queries extractions where `collectedAt` falls within today's UTC date range (00:00:00 to 23:59:59).
2. Queries extractions where `collectedAt` falls within yesterday's UTC date range.
3. For each day, deduplicates by `modelName` keeping the latest extraction per model (highest `collectedAt`).
4. Computes `newModels`: models in today's set but not in yesterday's set.
5. Computes `priceChanges`: models in both sets where `inputPricePer1m` or `outputPricePer1m` differs. Calculate `changePercent` as `((new - old) / old) * 100`.
6. Queries promotions where `createdAt` falls within today's UTC date range for `newPromotions`.
7. Sets `totalModelsToday` to the count of unique models in today's extractions.
8. Per D-05/Pitfall 3: When yesterday's extractions are empty, treat ALL today's extractions as `newModels`.

Use Drizzle's `and`, `gte`, `lt`, `eq`, `desc` operators. Import `db` from `@/src/db/index` and `extractions`, `promotions`, `sources` from `@/src/db/schema`.

Create `tests/pipeline/article-diff.test.ts` with tests covering:
- Returns all today's models as newModels when yesterday is empty (first-run case)
- Correctly identifies new models not present yesterday
- Correctly identifies price changes with accurate changePercent
- Returns empty results when both days have identical data
- Includes new promotions created today
- Handles the case where today has no extractions (all empty arrays, totalModelsToday = 0)
  </action>
