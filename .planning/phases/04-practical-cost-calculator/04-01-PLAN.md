<objective>
Define four practical cost scenarios with realistic token counts and implement cost calculation utilities that convert per-token pricing into real-world costs.

Purpose: Per COST-01 through COST-04, users need to see what AI models actually cost for common use cases. This plan creates the data layer: scenario definitions with token counts and pure calculation functions that compute costs from existing extraction pricing data. No UI yet -- that comes in Plan 2.

Output:
- app/lib/cost-scenarios.ts (scenario definitions)
- app/lib/pricing-utils.ts (calculation functions added)
- tests/pricing-utils.test.ts (calculation tests added)
</objective>
