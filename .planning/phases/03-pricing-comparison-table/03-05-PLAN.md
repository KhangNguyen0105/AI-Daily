## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser interaction -> client state | User-controlled filters, sorting, pagination, and currency state affect rendering. |
| External links -> provider sites | Source links navigate users away from AI Daily. |
| Responsive layout -> mobile users | Hidden columns must not hide required safety information without an alternate route. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-05-01 | Information Disclosure | External source links | mitigate | Keep provider-name link text only, source URL `href` and `title` tooltip, `target="_blank"`, and `rel="noopener noreferrer"`. |
| T-03-05-02 | Denial of Service | Mobile table usability | mitigate | Use horizontal scroll, fixed min widths, progressive column hiding, and manual viewport verification. |
| T-03-05-03 | Spoofing | Confidence display | mitigate | Preserve green/yellow/red color-only pill states with non-visible `aria-label` or `title`; do not add on-screen state labels or icons. |
| T-03-SC | Tampering | UI packages/registries | mitigate | Do not initialize shadcn, do not add `components.json`, do not install packages, and use standard Tailwind/local components only. |
</threat_model>
